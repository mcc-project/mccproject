Masternode Community Coin(MMC)
================

###	Coin Spec
	* Algorithm : x11 (POS/MN)
	* Coin Ticker : MCC
	* Max supply : 30,000,000
	* Block : 30 sec
	* Staking-time : 2hrs
	* MN collateral : 3,000 MCC (40% POS / 60% MN)
	* Premine : 300,000 MCC

###	Block reward
*	Block 1 = pre-mine
*	      2 -   2,880 =  1 MCC (POW)
*	  2,881 -  11,524 =  2 MCC (POS/MN)
*	 11,525 - 500,000 = 30 MCC (POS/MN)
*	500,001 -         =  5 MCC (POS/MN)

*	port 29868
*	rpc  29869
